Detailed Description

Класс для шифрования и дешифрования текста методом Виженера с поддержкой русского алфавита (через wchar_t).

Данный класс реализует классический шифр Виженера, который выполняет символьный сдвиг в пределах алфавита. Для русских букв используется 32-буквенный алфавит (без 'Ё').

Пример использования:

VigenereCipher cipher(L"КЛЮЧ");
std::wstring encrypted = cipher.zasifrovat(L"ПРИМЕР ТЕКСТА");
std::wstring decrypted = cipher.rasshifrovat(encrypted);
