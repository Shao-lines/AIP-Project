Detailed Description

Класс поворотного шифра решётки (Turning Grille Cipher).

Класс для шифрования и дешифрования текста методом поворотной решётки (Turning Grille Cipher).

Работает с квадратной решёткой чётного размера. Позволяет шифровать и дешифровать текст.

Решётка формируется с отверстиями и поворачивается по часовой стрелке. Работает с Unicode и поддерживает визуализацию состояния решётки и сетки.
