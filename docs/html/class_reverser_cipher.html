Parameters

Returns
Зашифрованный текст.

Текст разбивается на блоки фиксированного размера. Каждый блок инвертируется. Если включён режим уменьшения, размер блока уменьшается на 1 после каждого шага, но не может быть меньше 2.

Parameters
textИсходный текст для шифрования.
block_sizeНачальный размер блока.
shrinkingtrue — блоки уменьшаются, false — размер фиксирован.
Returns
Зашифрованный текст.
